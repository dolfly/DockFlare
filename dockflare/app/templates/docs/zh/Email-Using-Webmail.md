# 使用网页邮件 (PWA)

DockFlare 内置了现代化、响应式的网页邮件客户端，让您可以在任何设备上管理邮件。

## 访问网页邮件

登录网页邮件有两种方式：

1.  **SSO（单点登录）：** 如果您是已登录 DockFlare Master 界面的管理员，在邮件页面点击 **打开网页邮件**，系统将自动完成认证并登录您的邮箱。
2.  **直接登录：** 访问 `https://mail.yourdomain.com`。如果您在 Master 界面为邮箱设置了密码，可以直接使用电子邮件地址和密码登录。

## 安装为 PWA

DockFlare 网页邮件是一款**渐进式 Web 应用（PWA）**，可安装到您的设备上，获得接近原生应用的体验。

### 在移动端（iOS/Android）（移动端支持目前正在开发中）
*   在移动浏览器中打开网页邮件 URL。
*   **iOS：** 点击「分享」图标，选择 **添加到主屏幕**。
*   **Android：** 点击三点菜单，选择 **安装应用** 或 **添加到主屏幕**。

### 在桌面端（Chrome/Edge/Brave）
*   在地址栏查找「安装」图标（通常是一个带向下箭头的小显示器图标）。
*   点击 **安装**。

## 核心功能

*   **搜索：** 使用搜索栏查找邮件。DockFlare 利用全文检索（FTS5）在本地对主题、发件人和邮件正文建立索引。
*   **推送通知：** 在网页邮件设置中启用通知，即可在桌面或移动设备上实时收到新邮件提醒。

## 安全性

*   **EdDSA 认证：** 网页邮件为所有 API 交互使用由 DockFlare Master 签发的高安全性 Ed25519 JSON Web Token (JWT)。
*   **HTML 净化：** 所有传入的 HTML 邮件在渲染前均通过 DOMPurify 进行净化，防止跨站脚本攻击（XSS）和追踪像素。
