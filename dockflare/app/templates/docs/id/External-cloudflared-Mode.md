# Mode `cloudflared` Eksternal

DockFlare menyediakan mode lanjutan **External `cloudflared` Mode** bagi pengguna yang ingin mengelola agent `cloudflared` sendiri.

Topik ini dibahas lebih rinci pada halaman **[Internal vs. External `cloudflared`](Internal-vs-External-cloudflared.md)**.

Silakan buka halaman tersebut untuk penjelasan lengkap mengenai:
*   Perbedaan antara mode Internal dan External
*   Kelebihan dan kekurangan masing-masing pendekatan
*   Cara mengonfigurasi DockFlare agar memakai agent `cloudflared` eksternal
