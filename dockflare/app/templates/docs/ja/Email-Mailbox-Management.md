# メールボックスとクォータ管理

メールページの **メールボックス管理** カードでは、誰がメールを受信できるか、どれだけのストレージを使用できるかを制御します。

## メールボックスの作成

1.  **メールボックスを追加** をクリックします。
2.  **アドレス:** 希望のプレフィックスを入力します (例: `info`)。ドメインは自動的に付加されます。
3.  **表示名:** 受信者に表示される名前 (例: `サポートチーム`)。
4.  **クォータ:** 初期ストレージ制限を選択します。

## クォータシステムについて

DockFlare は、サーバーのディスク容量が不足しないようにしながら、ユーザーに適切なエクスペリエンスを提供するために、段階的なクォータシステムを使用しています。

### ソフトリミット (クォータ)
メールボックスが設定されたクォータを超えた場合:
*   システムはシステムアドレスからの**警告メール**をユーザーの受信ボックスに挿入します。
*   ユーザーはハードリミットに達するまでメールを受信し続けることができます。
*   マスター UI のクォータバーが**黄色**に変わります。

### ハードリミット (拒否)
ハードリミットは自動的に**ソフトリミット + 15% (最低 10 MB のグレースバッファ)** として計算されます。
*   **エッジでの拒否:** 拒否は Cloudflare エッジで発生します。送信者のメールサーバーは SMTP エラー **5.2.2 Mailbox full** を受け取ります。
*   メールは R2 トランジットバケットやローカルサーバーに一切届かず、帯域幅が節約されます。
*   マスター UI のクォータバーが**赤色**に変わります。

## キャッチオールメールボックス

キャッチオールメールボックスは、既存の特定のメールボックスと一致しない、ドメイン宛のすべてのメールを受信します。
1.  **キャッチオールを設定** をクリックします。
2.  宛先のメールボックスを選択します。
3.  **有効にする** をクリックします。

## 自動返信 (休暇モード)

任意のメールボックスに自動返信を設定できます。
1.  メールボックスの横にある **自動返信** アイコン (ロボット) をクリックします。
2.  メッセージと件名を入力します。
3.  返信機能が有効になる**日付範囲**を設定します。
4.  **返信間隔:** 「メールループ」を防ぐために、同じ送信者への返信頻度 (例: 24 時間に 1 回) を設定します。
