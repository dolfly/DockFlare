# ウェブメールを使う (PWA)

DockFlare には、あらゆるデバイスからメールを管理できる、モダンでレスポンシブなウェブメールクライアントが含まれています。

## ウェブメールへのアクセス

ウェブメールへのログイン方法は 2 通りあります。

1.  **SSO (シングルサインオン):** DockFlare マスター UI に管理者としてログインしている場合、メールページの **ウェブメールを開く** をクリックします。自動的に認証され、メールボックスにログインされます。
2.  **直接ログイン:** `https://mail.yourdomain.com` に移動します。マスター UI でメールボックスにパスワードを設定している場合、メールアドレスとパスワードで直接ログインできます。

## PWA としてインストール

DockFlare ウェブメールは **プログレッシブウェブアプリ (PWA)** です。デバイスにインストールして、アプリのような体験を得ることができます。

### モバイル (iOS/Android) の場合 (モバイルサポートは現在開発中)
*   モバイルブラウザでウェブメールの URL を開きます。
*   **iOS:** 「共有」アイコンをタップし、**ホーム画面に追加** を選択します。
*   **Android:** 三点メニューをタップし、**アプリをインストール** または **ホーム画面に追加** を選択します。

### デスクトップ (Chrome/Edge/Brave) の場合
*   アドレスバーに「インストール」アイコン (通常、下向き矢印の付いた小さなモニター) が表示されます。
*   **インストール** をクリックします。

## 主な機能

*   **検索:** 検索バーを使用してメールを検索します。DockFlare は全文検索 (FTS5) を使用して、件名、送信者、メッセージ本文をローカルでインデックス化します。
*   **プッシュ通知:** ウェブメールの設定で通知を有効にすると、デスクトップまたはモバイルデバイスで新着メールのリアルタイム通知を受け取ることができます。

## セキュリティ

*   **EdDSA 認証:** ウェブメールはすべての API インタラクションに、DockFlare マスターが発行する高セキュリティの Ed25519 JSON ウェブトークン (JWT) を使用します。
*   **HTML サニタイゼーション:** 受信したすべての HTML メールは、クロスサイトスクリプティング (XSS) およびトラッキングピクセルから保護するために、レンダリング前に DOMPurify でサニタイズされます。
