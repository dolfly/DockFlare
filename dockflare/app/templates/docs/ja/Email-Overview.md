# メールスイート概要

DockFlare Email は、既存の DockFlare インフラストラクチャ上に構築された、完全なセルフホスト型の主権メールシステムです。クラウドベースのメールサービスの利便性と、セルフホスティングのプライバシーおよびコントロールを両立させることを目的として設計されています。

## 主権メールのコンセプト

従来、セルフホストのメールサーバーは「自宅 IP のブラックリスト登録」という問題により運用が困難でした。住宅用 IP アドレスは主要なメールプロバイダーによってブロックされることがあります。DockFlare は Cloudflare を**ステートレスな配信ネットワーク**として活用することでこの問題を解決します。

1.  **Cloudflare** が SMTP 配信、MX ルーティング、および一時的なバッファリングという重要な役割を担います。
2.  **DockFlare** がデータを所有します。メッセージ、添付ファイル、メールボックスの設定はすべて自分のハードウェア上に保存されます。

Cloudflare のインフラストラクチャ上にメールの内容が永続的に保存されることはありません。トランジット中に一時的に R2 バケットにバッファリングされ、ローカルの Mail Manager が処理した直後に削除されます。

## アーキテクチャ

システムは複数の統合されたコンポーネントで構成されています。

*   **受信フロー:** インターネット → Cloudflare Email Routing → Inbound Worker → R2 バッファ → DockFlare Mail Manager Webhook → ローカルストレージ。
*   **送信フロー:** Webmail UI → Mail Manager API → Outbound Worker → Cloudflare `send_email` → インターネット。
*   **データ主権:** すべてのメールはローカルの SQLite データベースに解析・保存され、添付ファイルはローカルのファイルシステムに保存されます。

## 主な機能

*   **マルチドメイン対応:** Cloudflare で管理しているドメインの数だけメールをホストできます。
*   **エッジでのクォータ適用:** メールボックスが満杯の場合、Cloudflare Workers がサーバーに届く前に SMTP レベル (5.2.2) でメールを拒否し、帯域幅を節約します。
*   **全文検索:** SQLite FTS5 を使用して全メールを高速で検索できます。
*   **プライバシー優先:** すべての API インタラクションに EdDSA JWT 認証を使用。HTML メールコンテンツはレンダリング前にサニタイズされ、XSS およびトラッキングピクセルを防止します。
*   **PWA ウェブメール:** スマートフォンやデスクトップにインストールできる、モダンでモバイル対応のウェブメールクライアント。
*   **プッシュ通知:** Web Push (VAPID) を通じて新着メールをリアルタイムで通知します。
*   **耐障害性:** サーバーがオフラインになった場合でも、Cloudflare R2 が受信メールをバッファリングし、5 分ごとに自動的に配信を再試行します。
