# Mode `cloudflared` externe

DockFlare fournit un « mode `cloudflared` externe » avancé pour les utilisateurs qui souhaitent gérer eux-mêmes l'agent `cloudflared`.

Ce sujet est traité en détail sur la page **[Interne vs. Externe `cloudflared`](Internal-vs-External-cloudflared.md)**.

Veuillez vous référer à cette page pour une explication complète de :
* La différence entre les modes interne et externe.
* Les avantages et les inconvénients de chaque approche.
* Comment configurer DockFlare pour utiliser un agent `cloudflared` externe.
