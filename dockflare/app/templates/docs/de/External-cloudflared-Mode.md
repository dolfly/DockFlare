# Externer `cloudflared`-Modus

DockFlare bietet einen erweiterten "Externen `cloudflared`-Modus" für Benutzer, die den `cloudflared`-Agenten selbst verwalten möchten.

Dieses Thema wird ausführlich auf der Seite **[Interner vs. Externer `cloudflared`](Internal-vs-External-cloudflared.md)** behandelt.

Bitte lesen Sie auf dieser Seite die vollständige Erklärung zu:
*   Dem Unterschied zwischen dem internen und externen Modus.
*   Den Vor- und Nachteilen jedes Ansatzes.
*   Der Konfiguration von DockFlare zur Nutzung eines externen `cloudflared`-Agenten.
